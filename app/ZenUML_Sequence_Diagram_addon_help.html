Description

ZenUML Sequence Diagram is a Confluence macro add-on for creating sequence diagram based on pseudo-code with realtime rendering.

Usage

  1. Start by creating or editing a page in Confluence
  2. Insert the macro either
    • from the Insert more content menu
    • or by simply typing {zenuml sequence diagram
  3. Start creating your own sequence diagram using pseudo-code

Example

Note that you can drag the vertical bar in the middle to adjust the width of code editor.

Limitations

Export to PDF/Word is not supported at the moment.

Privacy Statement

ZenUML Sequence Diagram is a static macro. It doesn't store any user data outside of Confluence.
